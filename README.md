# Roll a Ball Game

A 3D Unity game inspired by the classic Roll-a-Ball concept, redesigned into a more complete mini-game with multiple levels, a main menu, collectibles, and sound effects.

## Overview

This project was developed in Unity using C#. The game focuses on controlling a ball through different environments, collecting items, and completing levels. Compared to the original prototype-style version, this version includes proper scene flow, improved visuals, and additional gameplay structure.

## Features

- Main menu
- 3 playable levels
- Collectible system
- Rotating collectible objects
- Sound effects
- Level progression
- Improved environment design

## Controls

- **Move:** WASD
- **Jump:** Space

## Built With

- Unity
- C#

## Project Structure

Main parts of the project include:
- Player movement using Rigidbody physics
- Collectible and score system
- Scene-based level progression
- UI for menu and gameplay
- Audio integration for feedback and game feel

## How to Play

Move the ball across the level, collect all required collectibles, and complete each stage. Be careful not to fall off the map.
